# mainm-aiyouxi-guide

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容涉及各类工具、信息整理与交互演示，主要用于个人学习与记录。本仓库不针对任何特定网站或服务，所有页面均为独立的静态文件。

## 目录说明

```
mainm-aiyouxi-guide/
├── index.html          # 仓库首页入口
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── README.md           # 本文件
```

- `index.html`：仓库的默认入口页面，可快速访问已归档的页面列表。
- `pages/`：所有独立 HTML 页面均存放于此目录下，按需命名与组织。

## 页面归档说明

本仓库中的 HTML 页面均为静态文件，不依赖后端服务或数据库。每个页面独立完成其功能或展示内容，可直接在浏览器中打开运行。页面内容可能包含工具、信息汇总、交互示例等，不涉及商业或推广性质的信息。
